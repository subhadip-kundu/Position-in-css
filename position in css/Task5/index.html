• The z-index property is a CSS property that controls the stacking order of elements in a web page. It specifies the z-axis position of an element in the stacking context, which determines how elements are layered on top of one another in a 2D space on the web page.
  • Elements with a higher z-index value will appear on top of elements with a lower value.
  • The z-index property only works on positioned elements, meaning the element must have a positioning value other than the default "static," such as "relative," "absolute," or "fixed."

In this example, we have three colored boxes, each with different z-index values. The CSS properties position: absolute; are applied to make the boxes positioned elements.

  • box1 has a z-index of 1.
  • box2 has a z-index of 2.
  • box3 has a z-index of 3.

As a result, box3 with a higher z-index value appears on top of box2, and box2 appears on top of box1. The stacking order is determined by the z-index values, and it is independent of the order in which the HTML elements appear in the code. If two elements have the same z-index, the one that appears later in the HTML source order will be on top.
